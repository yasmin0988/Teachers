# Teacher Units Tracker

A simple Python program that calculates teaching units for lecturers.

## What it does:
- Shows courses taught by each lecturer
-  Calculates total units per lecturer
-  Displays results in a clean table

## How to run:
```bash
python main.py
Example output:
